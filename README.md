## Run Locally

Clone the project

```bash
  git clone https://github.com/Athromic/athromic-t3.git
```

Go to the project directory

```bash
  cd atomic-t3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
